# Mathematics Curriculum — Formulas & Key Concepts Reference

---

## 1. Counting & Numbers

| Concept | Formula / Rule |
|---|---|
| Place Value | ones · 10⁰, tens · 10¹, hundreds · 10², thousands · 10³ … |
| Expanded Form | 3,472 = 3×1000 + 4×100 + 7×10 + 2×1 |
| Number Line | Distance between two integers a and b = |a − b| |

---

## 2. Addition & Subtraction

| Concept | Formula |
|---|---|
| Identity Property | a + 0 = a |
| Commutative | a + b = b + a |
| Associative | (a + b) + c = a + (b + c) |
| Break Apart (decompose) | 47 + 36 = (40+30) + (7+6) = 76 + 7 = 83 |
| Rounding (to nearest n) | If digit after place < 5 → round down; ≥ 5 → round up |
| Estimation | Round each number, then add/subtract |

**Rounding Rule:**
> Round 2,748 to nearest hundred → look at tens digit (4) → round down → **2,700**

---

## 3. Multiplication

| Concept | Formula |
|---|---|
| Repeated Addition | a × b = a + a + … (b times) |
| Identity | a × 1 = a |
| Zero Property | a × 0 = 0 |
| Commutative | a × b = b × a |
| Associative | (a × b) × c = a × (b × c) |
| Distributive | a × (b + c) = a×b + a×c |
| Partial Products | 23 × 14 = (20×10) + (20×4) + (3×10) + (3×4) |
| Area Model | Area = length × width |
| Estimation | Round factors, then multiply |

---

## 4. Division

| Concept | Formula |
|---|---|
| Basic Relationship | dividend ÷ divisor = quotient |
| Multiplication Link | a × b = c ↔ c ÷ a = b |
| Remainder | dividend = (divisor × quotient) + remainder |
| Long Division Check | divisor × quotient + remainder = dividend |
| Reciprocal | Reciprocal of a/b = b/a |
| Division with Reciprocal | a ÷ (b/c) = a × (c/b) |
| Estimate Quotient | Round dividend and divisor to compatible numbers |

---

## 5. Fractions

| Concept | Formula |
|---|---|
| Fraction Definition | p/q where p = numerator, q = denominator (q ≠ 0) |
| Unit Fraction | 1/q |
| Whole as Fraction | n = n/1 = nq/q |
| Equivalent Fractions | a/b = (a×k)/(b×k) for any k ≠ 0 |
| Comparing (same denom) | a/c vs b/c → compare a and b |
| Comparing (same numer) | a/b vs a/c → compare denominators (larger denom = smaller fraction) |
| Cross-multiply compare | a/b vs c/d → compare ad vs bc |
| Fractions on Number Line | a/b lies at a equal parts of size 1/b from 0 |

---

## 6. Addition & Subtraction of Fractions

| Concept | Formula |
|---|---|
| Like Fractions | a/c ± b/c = (a ± b)/c |
| Unlike Fractions | a/b ± c/d = (ad ± bc)/(bd) |
| LCD Method | Find LCM(b,d), convert both fractions, then add/subtract |
| Decompose | 7/10 = 3/10 + 4/10 = 0.3 + 0.4 |
| Tenths & Hundredths | a/10 = (a×10)/100; used for converting |

---

## 7. Mixed Fractions

| Concept | Formula |
|---|---|
| Mixed to Improper | a b/c = (a×c + b)/c |
| Improper to Mixed | p/q → divide p by q → quotient and remainder r → q_r/q |
| Add Mixed Fractions | Add whole parts + add fraction parts (regroup if needed) |
| Subtract Mixed Fractions | May need to regroup: a b/c = (a−1) (b+c)/c |
| Estimation | Round each mixed fraction to nearest whole, then compute |

---

## 8. Multiplication & Division of Fractions

| Concept | Formula |
|---|---|
| Multiply Fractions | a/b × c/d = (a×c)/(b×d) |
| Divide Fractions | a/b ÷ c/d = a/b × d/c = (a×d)/(b×c) |
| Multiply Mixed Fractions | Convert to improper, multiply, simplify |
| Divide Mixed Fractions | Convert to improper, apply division rule |
| Number × Fraction | n × a/b = (n×a)/b |
| Fraction × Number | a/b × n = (a×n)/b |
| Number ÷ Fraction | n ÷ a/b = n × b/a |

---

## 9. Decimals

| Concept | Formula |
|---|---|
| Place Values | tenths (10⁻¹), hundredths (10⁻²), thousandths (10⁻³) |
| Decimal to Fraction | 0.a = a/10; 0.ab = ab/100; 0.abc = abc/1000 |
| Fraction to Decimal | a/10 = 0.a; a/100 = 0.0a; a/1000 = 0.00a |
| Expanded Form | 3.47 = 3×1 + 4×(1/10) + 7×(1/100) |
| Comparing Decimals | Align decimal points; compare digit by digit left to right |
| Rounding Decimals | Same rule as integers; look at digit to the right of target place |

---

## 10. Addition & Subtraction of Decimals

| Concept | Formula |
|---|---|
| Standard Method | Align decimal points, add/subtract, keep decimal aligned |
| Break Apart | 3.47 + 2.15 = (3+2) + (0.47+0.15) = 5.62 |
| Estimation | Round decimals, then add/subtract |
| Regrouping | 10 hundredths = 1 tenth; 10 tenths = 1 whole |

---

## 11. Multiplication & Division of Decimals

| Concept | Formula |
|---|---|
| Decimal × Whole | Multiply ignoring decimal, then count decimal places |
| Decimal × Decimal | Multiply as integers; place decimal (sum of places in factors) |
| Decimal ÷ Whole | Divide normally; keep decimal point aligned |
| Decimal ÷ Decimal | Multiply both by 10^n to make divisor a whole, then divide |
| Area Model | Same as integers, account for decimal placement |
| Powers of 10 Rule | a × 10^n → move decimal n places right; a ÷ 10^n → move n places left |

## 14. Geometry — 1D, 2D, 3D

### Angles
| Type | Measure |
|---|---|
| Acute | 0° < θ < 90° |
| Right | θ = 90° |
| Obtuse | 90° < θ < 180° |
| Straight | θ = 180° |
| Reflex | 180° < θ < 360° |
| Full | θ = 360° |

### Triangle Classification
| By Sides | By Angles |
|---|---|
| Equilateral: 3 equal sides | Acute: all angles < 90° |
| Isosceles: 2 equal sides | Right: one angle = 90° |
| Scalene: no equal sides | Obtuse: one angle > 90° |

**Triangle Angle Sum:** ∠A + ∠B + ∠C = 180°

### Quadrilaterals
| Shape | Properties |
|---|---|
| Rectangle | 4 right angles; opposite sides equal |
| Square | 4 right angles; 4 equal sides |
| Rhombus | 4 equal sides; opposite angles equal |
| Parallelogram | Opposite sides parallel and equal |
| Trapezoid | Exactly one pair of parallel sides |

## 15. Geometry — Perimeter, Area, Volume

### Perimeter
| Shape | Formula |
|---|---|
| Rectangle | P = 2(l + w) |
| Square | P = 4s |
| Triangle | P = a + b + c |
| Parallelogram | P = 2(a + b) |
| Rhombus | P = 4a |

### Area
| Shape | Formula |
|---|---|
| Rectangle | A = l × w |
| Square | A = s² |
| Triangle | A = ½ × b × h |
| Parallelogram | A = b × h |
| Rhombus | A = ½ × d₁ × d₂ |
| Trapezoid | A = ½ × (a + b) × h |

### Volume
| Shape | Formula |
|---|---|
| Cube | V = s³ |
| Cuboid (Rectangular Prism) | V = l × w × h |
| Cylinder | V = π × r² × h |
| Cone | V = ⅓ × π × r² × h |

### Surface Area
| Shape | Formula |
|---|---|
| Cube | SA = 6s² |
| Cuboid | SA = 2(lw + lh + wh) |

## 17. Powers of 10

| Concept | Formula |
|---|---|
| Standard Form | 10^n = 1 followed by n zeros |
| Multiply by 10^n | Move decimal point n places right |
| Divide by 10^n | Move decimal point n places left |
| Scientific Notation | a × 10^n where 1 ≤ a < 10 |

**Powers Table:**
| Power | Value |
|---|---|
| 10⁰ | 1 |
| 10¹ | 10 |
| 10² | 100 |
| 10³ | 1,000 |
| 10⁴ | 10,000 |
| 10⁵ | 100,000 |
| 10⁶ | 1,000,000 |
| 10⁹ | 1,000,000,000 (billion) |
| 10¹² | 1,000,000,000,000 (trillion) |
